# Invoice to Purchase Order Extractor 📄🤖

An AI-powered data extraction pipeline that automates the parsing of invoices and purchase orders. This project leverages intelligent agents to extract structured data—such as buyer details, line items, and financial totals—from unstructured documents.

## ✨ Features

* **Automated Data Extraction:** Automatically pulls key invoice fields (Invoice Number, Date, GSTIN).
* **Buyer Details Parsing:** Extracts buyer name, address, and localized tax identifiers.
* **Line-Item Breakdown:** Captures tabular data including item descriptions, quantities, unit prices, and amounts.
* **Agentic Workflow:** Easily integrates with autonomous agents to automate document workflows end-to-end.
